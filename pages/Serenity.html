Serenity

Serenity es un chatbot para WhatsApp diseñado para facilitar la recopilación de información clínica, especialmente a través del Inventario de Depresión de Beck.
Este proyecto se realizó en colaboración con el CIC (Centro de Investigación en Cómputo - IPN).
El objetivo principal fue utilizar los datos recopilados para el entrenamiento de una Inteligencia Artificial.

Descripción General del Proyecto

La problemática

El M. en C. Abdiel Reyes Vera requería una solución eficiente para automatizar la recopilación de información clínica de manera confiable y oportuna, específicamente mediante el Inventario de Depresión de Beck. El proceso manual limitaba la capacidad de recolección, generaba inconsistencias y dificultaba la actualización sincronizada de las bases de datos en diferentes plataformas. Los principales retos fueron:

  • Ausencia de un sistema automatizado para recopilar y sincronizar información clínica entre MySQL y Firebase, limitando la capacidad de datos recopilados.
  • Riesgo de pérdida o desactualización de datos debido a procesos manuales y falta de sincronización en tiempo real.
  • Necesidad de mantener acceso actualizado y confiable para el entrenamiento continuo de modelos de Inteligencia Artificial.
  • Limitaciones técnicas para ejecutar, monitorear y garantizar la estabilidad del chatbot en producción.

La solución

Serenity es un chatbot para WhatsApp que automatiza la recopilación del Inventario de Depresión de Beck, garantizando la captura precisa y confiable de datos clínicos.

  • Desarrollo de un sistema de sincronización que integra periódicamente los datos almacenados en MySQL con Firebase, asegurando disponibilidad y actualización en tiempo real.
  • Uso de Hibernate para facilitar la gestión y persistencia de datos entre las bases MySQL y Firebase.
  • Implementación del bot en Node.js, gestionado con PM2 para garantizar su ejecución continua y monitoreo eficiente.
  • Automatización del flujo de recopilación de información, aumentando la capacidad y calidad de datos para su posterior uso en entrenamiento de modelos de inteligencia artificial.

Resultado e impacto

Serenity permitió automatizar la recopilación de datos clínicos, logrando un aumento significativo en la capacidad de recolección de información. En solo 2 meses se registraron más de 500 encuestas del Inventario de Depresión de Beck, superando ampliamente la capacidad del proceso manual anterior.

Implementación Técnica

ChatBot

Servidor

Chatbot desarrollado con Node.js y gestionado mediante PM2, capaz de manejar grandes volúmenes de información, almacenados eficientemente en una base de datos MySQL.

Sincronizador de datos

Implementación con Spring Data JPA para la interacción eficiente con MySQL, gestionando relaciones complejas entre entidades como trabajos, usuarios y asignaciones, además de aplicar estrategias de optimización y migraciones controladas.
